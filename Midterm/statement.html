Design Statement

The VantageAvenir project captures the fusion between storytelling, heritage, and travel design. This website is an exploration of visual clarity, usability, and emotional resonance.

Built with Flexbox, the design ensures adaptive layouts that remain cohesive across screens. Each page offers consistency in structure and accessibility—elements central to good user experience.

My goal was to balance elegant simplicity with narrative warmth—reflecting both modern web design principles and the authentic culture of Martinique.

Design Approach

Typography

Combines Montserrat for structure and Lora for warmth, emphasizing contrast and readability.

Color Palette

Uses dark greens and soft neutrals to evoke tropical depth and natural serenity.

Layout & Accessibility

Built with responsive Flexbox containers to ensure intuitive navigation and clear visual hierarchy.
